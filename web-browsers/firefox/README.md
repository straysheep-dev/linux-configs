
# Firefox

Preconfigured policy files for Firefox / Firefox-ESR, based on OpenSCAP policies.

Tested on:

- Ubuntu 18.04 - 24.04
- Kali 2020.1 - 2025.4
- Fedora 34
- Rocky 10

The README details are being maintained here now: [straysheep.dev/resources/#web-browsers](https://straysheep.dev/resources/#web-browsers).

For an offline copy:

```bash
# Just the resources.md file
wget https://github.com/straysheep-dev/straysheep-dev.github.io/raw/refs/heads/main/docs/resources.md

# Clone the full site
git clone https://github.com/straysheep-dev/straysheep-dev.github.io
```
